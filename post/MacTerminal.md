# Mac Terminal

获取系统最高权限 ：`sudo -s`； super user do

列出文件列表：`ls`;

带属性输出文件列表： `ls -l`

切换目录： `cd`+ 路径;

建立新文件：`touch filename`;

创建新文件空夹：`mkdir`;

拷贝文件：`cp -R /path/a /path/b`;

移动文件：`mv -R /path/a /path/b/`;

删除文件：`rm` `rm -rf`;

更改文件权限：`chmod`;

系统自带的清理脚本：`sudo periodic [time]` time:{daily,weekly,monthly}

清屏：`clear`

路由表的输出信息： `netstat -nr`
